# Azure Cloud Networking & Security Labs

This repository contains hands-on labs and study projects focused on 
Cloud Networking and Security using Microsoft Azure.

The labs are aligned with my preparation for the following certifications:
- Cisco CCNA
- Microsoft AZ-900
- Microsoft AZ-700
- LPIC-1 (Linux)

The goal is to apply networking fundamentals in cloud environments,
with emphasis on secure and scalable architectures.

## Skills covered
- TCP/IP and subnetting
- Azure Virtual Networks (VNets)
- Network Security Groups (NSG)
- VPN concepts
- Linux networking fundamentals
- Cloud security basics

## Tools & Technologies
- Microsoft Azure
- Azure Portal
- Azure CLI (basic)
- Linux (Ubuntu) in VM
- Cisco Packet Tracer
- Draw.io (network diagrams)
